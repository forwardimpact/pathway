# Metrics — Coaching

Suggested metrics for the coaching domain. These are starting points — discover
which metrics are most useful through practice. Record per the
[`kata-metrics`](../../kata-metrics/SKILL.md) protocol.

| Metric               | Unit  | Description                              | Data source |
| -------------------- | ----- | ---------------------------------------- | ----------- |
| meetings_facilitated | count | Storyboard meetings facilitated this run | Run actions |
| agents_participating | count | Domain agents present in the meeting     | RollCall    |
| experiments_active   | count | Open experiments on the storyboard       | Storyboard  |
| obstacles_identified | count | Obstacles listed after this meeting      | Storyboard  |

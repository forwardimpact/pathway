Outpost is installed

Open Outpost from /Applications/ to start the scheduler. It runs as a menu bar app — look for the tent icon.

Grant privacy access:

Outpost reads from Apple Mail, Calendar, and Contacts. macOS requires three separate grants on first launch — existing grants for prior Basecamp installs do not carry over because the bundle identifier changed:

1. Files & Folders — System Settings → Privacy & Security → Files & Folders. When macOS prompts for Documents / Mail / Calendar folders, grant access to each of these processes:

  • Outpost.app — the TCC responsible process (Swift launcher)
  • node — runs skill scripts with #!/usr/bin/env node shebangs
  • A version number like “2.1.72” — this is the Claude Code CLI. macOS shows its version string instead of a name. Safe to grant per-folder access.

2. Full Disk Access — System Settings → Privacy & Security → Full Disk Access. The sync-apple-mail and sync-apple-calendar skills read the macOS Mail and Calendar SQLite databases directly. Click + and add Outpost.app from /Applications/. Without this, calendar and mail sync fail silently.

3. Automation — System Settings → Privacy & Security → Automation. The draft-emails skill sends mail by driving Apple Mail through AppleScript. On first send attempt macOS will prompt “allow Outpost to control Mail” — click Allow.

If you are upgrading from a previous Outpost version and a process stops working, toggle its entry off then on again in the same panel — macOS revokes the permission when the binary changes.

If you are upgrading from Basecamp (bundle identifier renamed in v3.0.0), repeat all three grants for the new Outpost.app. Your old Basecamp grants are now orphaned and can be removed safely.

Next steps:

1. Edit your identity file:

vim ~/Documents/Personal/USER.md

2. Configure scheduled tasks:

vim ~/.fit/outpost/scheduler.json

3. Check scheduler status:

fit-outpost status

4. Stop the scheduler gracefully (before upgrading):

fit-outpost stop

5. Update skills in all knowledge bases (after upgrading):

fit-outpost update

6. Open your knowledge base interactively:

cd ~/Documents/Personal && claude

The installer runs update automatically on all configured knowledge bases. You can also run it manually at any time to push the latest skills.

Intel Macs: this .pkg ships only an Apple Silicon binary. On Intel hardware, install instead via the Homebrew cask: brew install --cask forwardimpact/tap/fit-outpost.

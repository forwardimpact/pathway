Basecamp is installed

Open Basecamp from /Applications/ to start the scheduler. It runs as a menu bar app — look for the tent icon.

Grant Files & Folders access:

Basecamp agents need access to specific folders (Documents, Mail, Calendar). When macOS prompts, grant only the folders each process needs via System Settings → Privacy & Security → Files & Folders:

  • Basecamp.app — the TCC responsible process (Swift launcher)
  • node — runs skill scripts with #!/usr/bin/env node shebangs
  • A version number like “2.1.72” — this is the Claude Code CLI. macOS shows its version string instead of a name. Safe to grant per-folder access.

If you are upgrading from a previous version and a process stops working, toggle it off then on again in Files & Folders — macOS revokes the permission when the binary changes.

Next steps:

1. Edit your identity file:

vim ~/Documents/Personal/USER.md

2. Configure scheduled tasks:

vim ~/.fit/basecamp/scheduler.json

3. Check scheduler status:

fit-basecamp --status

4. Stop the scheduler gracefully (before upgrading):

fit-basecamp --stop

5. Update skills in all knowledge bases (after upgrading):

fit-basecamp --update

6. Open your knowledge base interactively:

cd ~/Documents/Personal && claude

The installer runs --update automatically on all configured knowledge bases. You can also run it manually at any time to push the latest skills.

fit-doc X.Y.Z — Build and serve documentation sites from markdown

Usage: fit-doc <command> [options]

Commands:
  build  Build documentation site from markdown files
  serve  Build and serve documentation with optional watch mode

Options:
  --src=<string>  Source directory (default: public)
  --out=<string>  Output directory (default: dist)
  --help, -h      Show this help
  --version       Show version
  --json          Output help as JSON

Examples:
  fit-doc build
  fit-doc serve --watch --port 8080

Use fit-doc <command> --help for command-specific options.

# Step 3: Install the Forward Impact monorepo

Based on what you learned from the website documentation, install the Forward
Impact monorepo and its dependencies.

Follow the developer getting-started instructions:

1. Clone the monorepo from https://github.com/forwardimpact/monorepo.git into
   ./monorepo (inside the current working directory)
2. Run `bun install` to install all workspace dependencies
3. Verify the installation by checking that key binaries are available:
   - `bunx fit-pathway --help`
   - `bunx fit-map --help`

If bun is not installed, install it first.

# Step 1: Discover the Guide product

You are a developer evaluating the Forward Impact engineering framework
platform. Your goal is to understand the Guide product and how to install it in
a fresh project.

Start by reading the website at www.forwardimpact.team. Specifically:

1. Fetch and read the main docs page at https://www.forwardimpact.team/docs/
2. Fetch and read the Guide product page at
   https://www.forwardimpact.team/guide/
3. Fetch and read the developer getting-started page at
   https://www.forwardimpact.team/docs/getting-started/developers/
4. Fetch and read the "Finding Your Bearing" guide at
   https://www.forwardimpact.team/docs/guides/finding-your-bearing/

After reading all four pages, write a summary to ./notes/01-discovery.md:

- What Guide is and what it does
- What products and packages are available for installation
- The installation steps for developers (both direct install and monorepo)
- What data or configuration Guide needs to work
- Any gaps or unclear points in the documentation

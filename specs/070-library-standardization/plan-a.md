# Plan: 070 — Library Standardization and Skill Consolidation

This plan is split into three sequential parts. Each part builds on the
previous, but is independently executable and committable.

## Part index

| #   | Document                     | Scope                                             |
| --- | ---------------------------- | ------------------------------------------------- |
| 1   | [plan-a-01.md](plan-a-01.md) | Migrate non-conforming libraries to OO+DI         |
| 2   | [plan-a-02.md](plan-a-02.md) | Consolidate library skills into capability groups |
| 3   | [plan-a-03.md](plan-a-03.md) | Migrate products to OO+DI and integrate libraries |

## Execution order

Parts must be executed in order: part 01 standardizes libraries, part 02
consolidates skills (depends on the standardized libraries), part 03 extends the
pattern into products (depends on both prior parts).

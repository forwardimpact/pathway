# Brief — `fit-trace overview --filter-tools`

Spec a new `--filter-tools` flag for `fit-trace overview` that restricts the
output to events involving a comma-separated list of tool names (for example,
`--filter-tools=Bash,Edit` would limit the overview to events where the agent
invoked the `Bash` or `Edit` tools).

The hire is the persona+job at the level-2 heading of `specs/jtbd-excerpt.md`;
quote that heading verbatim in your spec's persona section.
